Netflix_Movie_Recommendation_System
End-to-end Machine Learning pipeline for predicting user movie preferences using Netflix rating history.
The project includes data preprocessing, handling missing values, duplicate removal, normalization, and encoding of user/movie IDs.
Feature engineering and selection techniques are applied using user behavior, movie popularity, genre, and historical rating patterns.
Classification models such as Logistic Regression, KNN, and SVM are trained to predict Recommended (1) or Not Recommended (0) movies.
Model performance is evaluated using accuracy, precision, recall, F1-score, confusion matrix, and ROC curve.
